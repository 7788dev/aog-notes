# 介绍

AOG Notes 是一个基于 Next.js 构建的现代化 Markdown 笔记框架，专为个人知识管理和内容分享而设计。

## 为什么选择 AOG Notes

- **零配置启动** - 克隆项目，添加 Markdown 文件，即可拥有一个专业的笔记网站
- **极致性能** - 基于 Next.js 静态生成，首屏加载极快
- **全文搜索** - 内置 BM25 搜索引擎，支持中文分词
- **响应式设计** - 完美适配桌面端和移动端
- **深色模式** - 支持浅色/深色/跟随系统三种模式
- **SEO 友好** - 自动生成 sitemap 和结构化数据

## 技术栈

| 技术 | 说明 |
|------|------|
| Next.js 16 | React 全栈框架 |
| Tailwind CSS 4 | 原子化 CSS 框架 |
| TypeScript | 类型安全 |
| React Markdown | Markdown 渲染 |
| Prism | 代码高亮 |

## 适用场景

- 个人技术博客
- 学习笔记整理
- 团队知识库
- 产品文档站点
- 课程教程网站
