# 安装

本指南将帮助你在本地环境中快速搭建 AOG Notes。

## 环境要求

- Node.js 18.0 或更高版本
- npm、yarn 或 pnpm 包管理器

## 安装步骤

### 1. 克隆项目

```bash
git clone https://github.com/your-username/easy-notes.git
cd easy-notes
```

### 2. 安装依赖

```bash
npm install
# 或
yarn install
# 或
pnpm install
```

### 3. 启动开发服务器

```bash
npm run dev
```

打开浏览器访问 `http://localhost:3000`，你应该能看到默认的笔记页面。

## 项目结构

```
easy-notes/
├── data/                 # Markdown 笔记目录
│   ├── 01-分类名称/
│   │   ├── 01-笔记.md
│   │   └── 02-笔记.md
│   └── 02-另一分类/
├── public/               # 静态资源
├── src/
│   ├── app/              # Next.js App Router
│   ├── components/       # React 组件
│   └── lib/              # 工具函数
├── site.config.ts        # 网站配置
└── package.json
```
