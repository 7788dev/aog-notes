# 自定义样式

AOG Notes 使用 Tailwind CSS 构建，你可以轻松自定义样式。

## 修改主题色

最简单的方式是在 `site.config.ts` 中修改 `themeColor`：

```typescript
const siteConfig = {
  themeColor: '#8b5cf6',  // 紫色主题
}
```

主题色会自动应用到：

- 侧边栏选中状态
- 搜索框聚焦边框
- 笔记列表选中状态
- 链接悬停颜色
- 代码块标识
- 引用块边框
- 二级标题左边框

### 推荐主题色

| 颜色 | 色值 | 风格 |
|------|------|------|
| 绿色 | `#16b777` | 清新自然（默认） |
| 蓝色 | `#3b82f6` | 专业稳重 |
| 紫色 | `#8b5cf6` | 优雅神秘 |
| 橙色 | `#f97316` | 活力热情 |
| 红色 | `#ef4444` | 醒目强烈 |
| 青色 | `#06b6d4` | 清爽现代 |

## 全局样式

全局样式文件位于 `src/app/globals.css`：

```css
@import "tailwindcss";
@plugin "@tailwindcss/typography";

/* 自定义滚动条 */
* {
  scrollbar-width: thin;
  scrollbar-color: #e5e7eb transparent;
}

/* 深色模式滚动条 */
.dark * {
  scrollbar-color: #374151 transparent;
}

/* 主题切换过渡 */
.theme-transition * {
  transition: none !important;
}
```

## 通过配置注入样式

在 `site.config.ts` 中使用 `customCode.css`：

```typescript
customCode: {
  css: `
    /* 自定义标题样式 */
    .prose h2 {
      font-size: 1.5rem;
      color: #1e40af;
    }
    
    /* 自定义代码块 */
    .prose pre {
      border-radius: 12px;
    }
    
    /* 自定义引用块 */
    .prose blockquote {
      font-style: normal;
      background: #f0f9ff;
    }
  `,
}
```

## Markdown 样式

Markdown 渲染样式在 `src/components/NoteReader.tsx` 中定义，使用 Tailwind Typography 插件：

```tsx
<article className="
  prose prose-slate dark:prose-invert max-w-none
  
  /* 标题样式 */
  prose-h2:text-lg prose-h2:mt-10 prose-h2:mb-5
  prose-h2:pl-3 prose-h2:border-l-[3px] prose-h2:border-[var(--theme-color)]
  
  /* 段落样式 */
  prose-p:text-[15px] prose-p:leading-8 prose-p:my-5
  
  /* 代码样式 */
  prose-code:bg-gray-100 dark:prose-code:bg-gray-800
  prose-code:text-[#d63384]
  
  /* 表格样式 */
  prose-table:text-[15px] prose-table:my-8
">
```

### 常用样式类

| 类名 | 说明 |
|------|------|
| `prose-h1:*` | 一级标题样式 |
| `prose-h2:*` | 二级标题样式 |
| `prose-p:*` | 段落样式 |
| `prose-a:*` | 链接样式 |
| `prose-code:*` | 行内代码样式 |
| `prose-pre:*` | 代码块样式 |
| `prose-blockquote:*` | 引用块样式 |
| `prose-table:*` | 表格样式 |
| `prose-li:*` | 列表项样式 |

## 代码高亮主题

代码高亮使用 Prism，支持浅色和深色两套主题：

```tsx
import { oneLight, oneDark } from 'react-syntax-highlighter/dist/esm/styles/prism'

// 根据当前主题选择
const isDark = document.documentElement.classList.contains('dark')
const theme = isDark ? oneDark : oneLight
```

### 可用主题

可以替换为其他 Prism 主题：

- `atomDark` - Atom 深色
- `dracula` - Dracula 主题
- `materialDark` - Material 深色
- `materialLight` - Material 浅色
- `nord` - Nord 主题
- `synthwave84` - Synthwave 风格
- `vscDarkPlus` - VS Code 深色
- `vs` - VS Code 浅色

修改 `src/components/NoteReader.tsx` 中的导入即可。

## 布局调整

### 侧边栏宽度

在 `src/components/HomeClient.tsx` 中修改：

```tsx
<aside className="w-52 ...">  {/* 默认 208px */}
```

常用宽度：
- `w-48` - 192px（紧凑）
- `w-52` - 208px（默认）
- `w-56` - 224px
- `w-64` - 256px（宽松）

### 笔记列表宽度

```tsx
<div className="w-full md:w-80 ...">  {/* 默认 320px */}
```

### 内容区最大宽度

```tsx
<div className="max-w-3xl mx-auto ...">  {/* 默认 768px */}
```

可选值：
- `max-w-2xl` - 672px
- `max-w-3xl` - 768px（默认）
- `max-w-4xl` - 896px
- `max-w-5xl` - 1024px

## 字体配置

默认使用 Geist 字体，在 `src/app/layout.tsx` 中配置：

```tsx
import { Geist, Geist_Mono } from "next/font/google";

const geistSans = Geist({
  variable: "--font-geist-sans",
  subsets: ["latin"],
});

const geistMono = Geist_Mono({
  variable: "--font-geist-mono",
  subsets: ["latin"],
});
```

### 使用其他字体

```tsx
import { Inter, JetBrains_Mono } from "next/font/google";

const inter = Inter({
  variable: "--font-sans",
  subsets: ["latin"],
});

const jetbrainsMono = JetBrains_Mono({
  variable: "--font-mono",
  subsets: ["latin"],
});
```

### 使用中文字体

```tsx
import { Noto_Sans_SC } from "next/font/google";

const notoSansSC = Noto_Sans_SC({
  variable: "--font-sans",
  subsets: ["latin"],
  weight: ["400", "500", "700"],
});
```

## 响应式设计

AOG Notes 使用 Tailwind 的响应式断点：

| 断点 | 最小宽度 | 说明 |
|------|----------|------|
| `sm` | 640px | 小屏手机 |
| `md` | 768px | 平板 |
| `lg` | 1024px | 笔记本 |
| `xl` | 1280px | 桌面 |
| `2xl` | 1536px | 大屏 |

示例：

```tsx
<div className="hidden md:block">  {/* 平板及以上显示 */}
<div className="w-full lg:w-80">   {/* 笔记本及以上固定宽度 */}
```

## 深色模式适配

使用 `dark:` 前缀添加深色模式样式：

```tsx
<div className="bg-white dark:bg-gray-900">
<p className="text-gray-600 dark:text-gray-300">
```

在自定义 CSS 中：

```css
.my-class {
  background: white;
}

.dark .my-class {
  background: #1f2937;
}
```
