# 功能说明

本文详细介绍 AOG Notes 的各项功能及使用方法。

## 全文搜索

### 使用方式

- 点击顶部搜索框输入关键词
- 使用快捷键 `Ctrl + K`（Mac: `Cmd + K`）快速聚焦搜索框
- 支持中文、英文及混合搜索

### 搜索特性

- 基于 BM25 算法，结果按相关性排序
- 标题匹配权重高于正文
- 支持模糊匹配
- 搜索结果实时高亮显示

## 收藏功能

### 使用方式

- 在文章详情页点击「收藏」按钮
- 侧边栏点击「我的收藏」查看所有收藏
- 再次点击可取消收藏

### 数据存储

收藏数据保存在浏览器本地存储（localStorage）中，清除浏览器数据会丢失收藏记录。

## 深色模式

### 切换方式

点击顶部的主题切换按钮，支持三种模式：

1. **浅色模式** - 白色背景
2. **深色模式** - 深色背景
3. **跟随系统** - 自动跟随操作系统设置

### 记忆功能

主题选择会保存在本地，下次访问自动应用。

## 文章导航

启用 `features.articleNav` 后，文章底部会显示上一篇/下一篇导航按钮，方便连续阅读。

## 面包屑导航

文章详情页顶部会显示导航路径：

```
首页 / 分类名称
```

点击可快速跳转到对应页面。

## RSS 订阅

启用 `features.rss` 后，系统自动生成 RSS 订阅源：

- 订阅地址：`/feed.xml`
- 页脚会显示 RSS 链接
- 支持所有标准 RSS 阅读器

## 阅读时间

启用 `features.readingTime` 后，文章详情页会显示预估阅读时间：

- 默认阅读速度：200 字/分钟
- 可在 `advanced.wordsPerMinute` 中调整
- 同时统计中文和英文字符

## 标签系统

### 添加标签

在 Markdown 文件的 Front Matter 中添加：

```yaml
---
tags:
  - JavaScript
  - 前端
  - 教程
---
```

### 使用标签

- 标签显示在文章详情页标题下方
- 点击标签自动搜索相关文章
- 支持多个标签

## PWA 支持

启用 `features.pwa` 后，网站支持：

### 安装到桌面

- 移动端：浏览器菜单 → 添加到主屏幕
- 桌面端：地址栏右侧安装图标

### 离线访问

已访问过的页面可以离线查看（需要浏览器支持 Service Worker）。

### 图标配置

在 `public/` 目录放置图标文件：

- `icon-192.svg` 或 `icon-192.png`（192x192）
- `icon-512.svg` 或 `icon-512.png`（512x512）

## SEO 优化

启用 `seo.enabled` 后，系统自动：

- 生成 sitemap.xml
- 添加 Open Graph 标签
- 添加 Twitter Card 标签
- 生成结构化数据（JSON-LD）
- 设置 canonical URL

### 站点验证

支持 Google 和百度站点验证：

```typescript
seo: {
  googleVerification: 'your-code',
  baiduVerification: 'your-code',
}
```

## 自定义代码

### 添加统计代码

```typescript
customCode: {
  headScript: `
    <!-- Google Analytics -->
    <script async src="https://www.googletagmanager.com/gtag/js?id=G-XXXXXX"></script>
    <script>
      window.dataLayer = window.dataLayer || [];
      function gtag(){dataLayer.push(arguments);}
      gtag('js', new Date());
      gtag('config', 'G-XXXXXX');
    </script>
  `,
}
```

### 自定义样式

```typescript
customCode: {
  css: `
    /* 自定义标题颜色 */
    .prose h2 {
      color: #3b82f6;
    }
    
    /* 自定义代码块背景 */
    .prose pre {
      background: #1e1e1e;
    }
  `,
}
```

## 键盘快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl/Cmd + K` | 聚焦搜索框 |
| `↑` / `↓` | 切换笔记 |
| `Esc` | 返回列表 |
